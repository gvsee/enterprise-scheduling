Block scheduling editor
Reserve time windows for specific patient types · click a cell to reassign · changes apply from next week
What is block scheduling? Each time window is reserved for a specific patient type — e.g. 9–11am is New Patients only, 11am–1pm is Follow-ups only, 2–3pm is Urgent / Same-day. Patients outside the block type see the slot as unavailable. Agents and admins can override. This prevents new patient slots from being consumed by follow-ups, improving throughput and predictability.
